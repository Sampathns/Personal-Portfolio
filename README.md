# Sampath N S — Personal Portfolio

A dark-themed personal portfolio showcasing AI/ML projects, built with Node.js + Express.

## Tech Stack
- **Server:** Node.js + Express
- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Fonts:** Syne + DM Sans (Google Fonts)
- **Icons:** Tabler Icons

## Local Development

```bash
npm install
npm start
# Open http://localhost:3000
```

## Deploy to Render

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New → Web Service
3. Connect your GitHub repo
4. Set the following:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Environment:** Node
5. Click **Deploy** — you're live! 🚀

## Customise

- Edit `public/index.html` to update your name, bio, email, projects, and skills.
- Replace `your@email.com` in the contact section with your real email.
